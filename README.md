# Muyang Lyu

Personal academic website for Muyang Lyu.

Built with Next.js and adapted from PRISM.

## Content

Main content lives in `content/`.

- `content/config.toml`: site profile, social links, navigation
- `content/bio.md`: about text
- `content/education.toml`: education timeline
- `content/publications.bib`: publications
- `content/cv.md`: CV page
